# SendTelegram

Projeto de implementação de uma solução de envio de mensagens de texto, arquivos e imagens através de um bot Telegram, utilizando a biblioteca Telegram.Bot e a linguagem C# para gerenciamento das funcionalidades e integração com a API do Telegram.

## 🚀 Instruções

Abaixo link com uma documentação explicando como configurar o projeto com seus dados.

* .Net Core 6 
* Biblioteca Telegram.Bot
